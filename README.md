# A/A/B Test Analysis
This project analyzes the results of an A/A/B test for a company that sells food products through an app. This test is about implementing a whole new font for the app.
The objective is to study the sales funnel to understand customers behaviour thourgh every interaction so we can discover the real impact of the font change. 

## Objectives
- Analyze user behaviour accross the sales funnel
- Identify if there is a signifcant statistical difference between the control groups (246/247) to understand if they were correctly divided through a **z-test**.
- Comparing the experimental group results with the control groups results.

## Techonologies Used
- Python
- Pandas
- Matplotlib
- Dataset: (logs.csv)
